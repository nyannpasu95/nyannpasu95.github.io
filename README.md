## Projects

### MagicTapper - Magic Mouse Utility
**MagicTapper** is a macOS utility designed to bring **trackpad-style tap-to-click functionality** to the **Apple Magic Mouse**.

It enhances the user experience by allowing touch gestures on the mouse surface, making the Magic Mouse feel more like a MacBook trackpad. This tool is perfect for users looking to customize their input device for better productivity.
